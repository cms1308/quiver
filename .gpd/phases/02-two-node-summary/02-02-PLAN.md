  <name>Task 1: Run exhaustive physics consistency checks</name>
  <files>results/two_node_tables.py</files>
  <action>
    Add a validation section to results/two_node_tables.py that performs ALL of the following checks on every class:

    Check 1 -- Hofman-Maldacena bounds:
    For each of the 316 classes with a/c defined: verify 1/2 <= a/c <= 3/2.
    For ALL 326 classes: verify a/N^2 >= 0 and c/N^2 >= 0.
    Record any violations with class_id, gauge_pair, rank sector, and actual values.

    Check 2 -- Exact vs numerical consistency:
    For each of the 181 classes with exact symbolic data (a_exact, c_exact in DB):
    - Parse a_exact and c_exact as sympy expressions
    - Evaluate to float
    - Compare with stored numerical a_over_N2 and c_over_N2
    - Flag any |exact - numerical| > 1e-8

    Check 3 -- c-a relation:
    The identity c - a = -(1/16)(Tr R - Tr R^3) provides an independent check.
    For classes with exact symbolic R-charges:
    - Parse R_exact strings to extract individual R-charge values
    - Compute Tr R and Tr R^3 from R-charges and field multiplicities
    - Verify c_over_N2 - a_over_N2 = -(1/16)(Tr_R - Tr_R3)
    If R-charge parsing is too fragile, fall back to: verify c = (1/32)(9*Tr_R3 - 5*Tr_R) and a = (3/32)(3*Tr_R3 - Tr_R) give the same c-a as stored values, using the representative theory's numerical R-charges.

    Check 4 -- Positivity and sanity:
    - No class has a_over_N2 < -1e-10 (only Type-I a=0 allowed to be near zero)
    - No class has c_over_N2 < -1e-10
    - For Type-I classes: |a_over_N2| < 1e-8 AND |c_over_N2| < 1e-8 (both genuinely zero)
    - a_over_c values form a continuous distribution (no wild outliers beyond [0.5, 1.5])

    Check 5 -- Theory count consistency:
    Sum of n_theories across all classes = total classified theories in theory table.
    7757 total theories - 220 unclassified = expected sum.

    Report: print summary of checks run, pass/fail counts, and any flagged items.
  </action>
  <verify>
    1. Zero Hofman-Maldacena violations
    2. Zero exact-numerical mismatches > 1e-8
    3. c-a relation holds for all tested classes
    4. No negative central charges
    5. Theory counts consistent
    6. All checks run on ALL classes (not a subset)
  </verify>
